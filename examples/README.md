# Examples

Setup: `npm run build && npm run setup-examples` from the project root.

Three examples showing different ways to use svelte-docinfo:

| Directory          | Description                                                        |
| ------------------ | ------------------------------------------------------------------ |
| [`vite/`](./vite/) | Vite plugin with HMR — recommended for Vite and SvelteKit projects |
| [`api/`](./api/)   | Direct API usage — standalone scripts and build tool integration   |
| [`cli/`](./cli/)   | CLI usage — quick analysis from the command line                   |

If you're using Vite or SvelteKit, start with `vite/`.
For standalone scripts or build tool integration, see `api/`.
For a quick look at what svelte-docinfo produces, try `cli/`.
